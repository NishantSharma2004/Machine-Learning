## 🛠️ General Installation & Running Guide

Each project has its own dedicated directory containing its source code, trained models, notebooks, and run instructions.

To run any of the web applications locally:

1.  **Clone this repository:**
    ```bash
    git clone https://github.com/NishantSharma2004/Machine-Learning.git
    cd Machine-Learning
    ```

2.  **Navigate to your target project folder:**
    *   *For Crop Recommendation:* `cd Crop_Recommendation`
    *   *For Laptop Price Prediction:* `cd Laptop_Price_Prediction`
    *   *For Used Bike Price Prediction:* `cd Used_Bike_Price_Prediction`

3.  **Install the specific dependencies:**
    ```bash
    pip install -r requirements.txt   # (Or install flask/streamlit, joblib, scikit-learn, pandas, numpy)
    ```

4.  **Launch the app:**
    *   *For Flask apps:* `python app.py`
    *   *For Streamlit apps:* `streamlit run app.py`
